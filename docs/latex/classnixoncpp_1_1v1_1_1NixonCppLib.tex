Constructs a \doxylink{classnixoncpp_1_1v1_1_1NixonCppLib}{Nixon\+Cpp\+Lib} instance from the given application context. 

Initializes the library by setting up the logger and asset manager from the provided application context. If no logger is supplied in the context, a Null\+Logger is used as a fallback. The constructor validates the asset manager and sets the initialization flag accordingly.


\begin{DoxyParams}{Parameters}
{\em context} & The application context containing the logger and asset manager dependencies required for library initialization.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
If the asset manager is null or fails validation, an error is logged and the library remains uninitialized (is\+Initialized\+\_\+ will be false). 
\end{DoxyNote}
